### 项目简介

---

 * **本项目作为学习使用**
    
    * 项目介绍
        
        项目使用SpringBoot+Mybatis+Shiro+Redis, 只作为学习使用,
        使用Alibaba代码规范,有常用工具类.

    * 已完成功能

        * Shiro权限验证(权限设计: 用户关联(部门+角色),角色关联(部门+权限),权限关联资源)
        * 日志收集
        * 文件上传
        * Swagger2
        * Activiti工作流
        * Elasticsearch7.6.x全文搜索
        * Quartz定时器
        * 视频转换
        * 多线程
        * 操作摄像头javacv

    * 待完成功能
        
        * jenkins, kubernetes
        * Redis分布式缓存
        * 消息队列 RabbitMQ Kafka
        * jvm调优
        * SpringCloud
